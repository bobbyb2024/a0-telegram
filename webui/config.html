Elevated Mode (2FA)
When enabled, authenticated users can unlock full Agent Zero access from Telegram using !auth <key>. Default mode is restricted (chat only, no tools).
SECURITY NOTICE
Elevated mode grants Telegram users full access to Agent Zero's capabilities, including code execution, file operations, and system tools. Before enabling, you must read and understand the security section in the plugin's README.md.

Recommended configuration: A private chat or group with only trusted members, a defined User Allowlist above, and short session timeouts. Only allow users you deeply trust.
